## Why this exists

The code is Fair Source: you may run it, study it, modify it, and (subject to
the FSL's no-Competing-Use terms) build on it. The **brand** is how users tell
the official, supported Concord apart from forks and repackagers. Keeping the
name and logo reserved lets anyone work with the code while protecting users
from "official-looking" knock-offs.
